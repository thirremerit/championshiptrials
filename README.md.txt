City Care

City Care is a web application designed to help citizens report city issues and allow government workers to manage and track them efficiently.

User Roles

There are two types of users:

Citizen – A regular user who can report issues in the city.

Government Worker (Admin) – An admin user who has additional privileges, such as changing the status of an issue to Pending, In Progress, or Resolved.

When signing up, users can choose their role.

Features

Report Issues: Users can submit issues and place a marker on the map to show the exact location of the problem.

View Issues: All reported issues can be viewed in a list and previewed on a map.

Filtering: Issues can be filtered by status to make it easier to track specific types of reports.

Admin Controls: Government workers can update the status of issues to indicate whether they are pending, in progress, or resolved.
